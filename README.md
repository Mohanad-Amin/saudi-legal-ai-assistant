⚖️ المستشار القانوني الذكي (بنية AWS المتكاملة)
تطبيق ويب متكامل (Full-Stack) وجاهز للإنتاج، يعمل كمساعد قانوني ذكي متخصص في القوانين السعودية. تم بناء المشروع على بنية سحابية قوية وقابلة للتوسع باستخدام خدمات Amazon Web Services (AWS)، ويتميز بآلية RAG متقدمة مع إعادة ترتيب المصادر (Re-ranking) لضمان أعلى مستويات الدقة.



✨ أبرز الميزات
بنية سحابية متكاملة (AWS): التطبيق منشور على AWS Elastic Beanstalk، مع استخدام S3 لتخزين الأصول و DynamoDB لإدارة بيانات المحادثات، مما يضمن أداءً عالياً وتوافرية وقابلية للتوسع.

سجل محادثات دائم (Persistent): يتم تخزين جميع المحادثات بشكل آمن في قاعدة بيانات DynamoDB، مما يتيح للمستخدمين الرجوع إلى محادثاتهم السابقة في أي وقت عبر رابط فريد.

آلية RAG متقدمة مع Re-ranking: يستخدم آلية متطورة لإعادة ترتيب المصادر ديناميكيًا، مما يزيد من دقة الإجابات في المحادثات المعقدة والمتابعة.

نشر تلقائي (CI/CD): تم إعداد مسار نشر متكامل باستخدام GitHub Actions يقوم تلقائيًا بنشر أي تحديثات على الفرع الرئيسي إلى بيئة AWS Elastic Beanstalk.

نظام لجمع آراء المستخدمين (Feedback System): واجهة تتيح للمستخدمين تقييم الإجابات، ويتم تخزين هذه التقييمات في DynamoDB لتحليلها وتحسين النظام مستقبلاً.

نموذج تضمين مُخصص: يعتمد على نموذج TheMohanad1/Fine-Tuned-E5 المُدرب خصيصًا على نصوص قانونية عربية لضمان فهم دقيق للاستعلامات.

يوضح الرسم التالي تدفق البيانات في النظام بشكل متكامل، من المستخدم وحتى الحصول على الإجابة عبر خدمات AWS المختلفة

المستخدم (User): يتفاعل مع الواجهة الأمامية (HTML, CSS, JS) من خلال المتصفح.

AWS Elastic Beanstalk: يستقبل طلبات HTTP ويوجهها إلى تطبيق Flask الذي يعمل على خوادم EC2 مُدارة.

تطبيق Flask (Backend):

عند بدء التشغيل، يقوم بتحميل نموذج الذكاء الاصطناعي (Fine-Tuned-E5) والبيانات القانونية من Amazon S3.

يتواصل مع Amazon DynamoDB لقراءة سجل المحادثات السابقة وكتابة الرسائل الجديدة وتقييمات المستخدمين.

ينفذ منطق RAG و Re-ranking المتقدم.

يتصل بواجهة برمجة تطبيقات OpenAI (GPT-4o) لتوليد الإجابة النهائية.

العودة للمستخدم: يتم إرسال الإجابة النهائية إلى الواجهة الأمامية لعرضها للمستخدم.

🛠️ التقنيات المستخدمة
الواجهة الخلفية (Backend): Python, Flask

الواجهة الأمامية (Frontend): HTML, Tailwind CSS, JavaScript (Fetch API)

نماذج الذكاء الاصطناعي:

Retriever: TheMohanad1/Fine-Tuned-E5

Generator: OpenAI gpt-4o

البنية التحتية السحابية (Cloud Infrastructure):

الحوسبة (Compute): AWS Elastic Beanstalk

التخزين (Storage): Amazon S3

قاعدة البيانات (Database): Amazon DynamoDB (NoSQL)

إدارة الهوية (Identity): AWS IAM

النشر التلقائي (CI/CD): GitHub Actions

🧠 ضبط نموذج التضمين (Fine-Tuning)
(هذا القسم يبقى كما هو لأنه لا يزال دقيقاً وممتازاً)

لتحقيق أقصى دقة في فهم اللهجة السعودية والنصوص العربية، تم عمل Fine-tuning لنموذج التضمين (TheMohanad1/Fine-Tuned-E5) انطلاقًا من النموذج الأساسي e5-multilingual-large. تمت عملية التدريب كالتالي:

مجموعة البيانات: تم استخدام أكثر من 10,000 نص قضائي باللغة العربية.

منهجية التدريب: لكل نص، تم توليد سؤال افتراضي يحاكي استفسارات المستخدمين الحقيقيين بتحديات متنوعة (أسئلة بالعامية، بالفصحى، وبأخطاء إملائية).

دالة الخسارة (Loss Function): تم استخدام MultipleNegativesRankingLoss.

التقييم: حقق النموذج أداءً متميزًا على مقياس Recall@10، حيث تجاوزت دقته 95%.

🚀 كيفية التشغيل محلياً
استنسخ المستودع:

git clone [ضع رابط المستودع هنا]
cd [اسم المجلد]

أنشئ وشغّل البيئة الافتراضية:

python3 -m venv venv
source venv/bin/activate
# For Windows: venv\Scripts\activate

ثبّت المكتبات المطلوبة:

pip install -r requirements.txt

إعداد المفاتيح السرية:

أنشئ ملفًا جديدًا باسم .env.

أضف مفتاح OpenAI الخاص بك: OPENAI_API_KEY="sk-..."

(اختياري للتشغيل المحلي الكامل) قم بإعداد متغيرات بيئة AWS إذا كنت ترغب في الاتصال بـ S3 و DynamoDB من جهازك.

شغّل تطبيق Flask:

python app.py

يمكنك الآن الوصول للتطبيق عبر http://127.0.0.1:5000.
